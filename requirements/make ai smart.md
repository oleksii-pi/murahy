Make AI smart.

In the first 3 minutes, it should focus on growing resource consumption. Feed the queen if needed. Create as many workers as possible. Upgrade the anthill a few times to enable the queen to produce more workers (when it is appropriate). Assign all new workers to collect food and send a few to collect materials. Try to calculate the build order for optimized growth. The goal is to increase the momentum of resource consumption and spending.

After 3 minutes, start to build guardians. Spread the guardians across 3 different points. Protect the queen with one group, while the other two should be ready to attack. After 10 minutes, with a good number of guardians, the AI can go and attack.

If I start an attacking in the first 3 minutes, nobody protects the enemy queen. Instead, the queen should be able to attack enemies; in 2 hits, it can kill one guardian. Additionally, workers can protect the queen if she is attacked.

Detect if there are too many workers and they are interfering with each other while feeding the larvae.
